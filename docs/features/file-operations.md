## 文件操作配置

### 操作权限
```json
{
  "fileOperations": {
    "allowRead": true,
    "allowWrite": true,
    "allowDelete": false,
    "requireConfirmation": true
  }
}
```

### 文件过滤
```json
{
  "fileFilter": {
    "allowedExtensions": [".js", ".py", ".md"],
    "blockedExtensions": [".exe", ".dll"],
    "maxFileSize": "10MB"
  }
}
```
