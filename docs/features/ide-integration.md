# IDE集成功能

五彩代码提供了与各种IDE的集成功能，让您可以在熟悉的开发环境中使用AI编程助手。

## 支持的IDE

### Visual Studio Code
- 安装VSCode扩展
- 直接在工作区中使用
- 支持文件操作和代码编辑
- 集成调试功能

### JetBrains系列
- IntelliJ IDEA
- WebStorm
- PyCharm
- 其他JetBrains IDE

### 其他编辑器
- Sublime Text
- Atom
- Vim/Neovim
- Emacs

## VSCode集成

### 安装扩展
1. 打开VSCode
2. 搜索"Wucai Code"
3. 点击安装
4. 重启VSCode

### 基本使用
```bash
# 在VSCode终端中启动
wucai

# 或者使用命令面板
Ctrl+Shift+P -> "Wucai Code: Run"
```

### 功能特性
- 文件自动同步

## 配置和设置

### IDE配置
```json
{
  "ide": {
    "type": "vscode",
    "autoSync": true,
    "showDiffs": true,
    "enableDebugging": true,
    "workspacePath": "./"
  }
}
```

### 集成设置
```json
{
  "integration": {
    "fileWatcher": true,
    "autoSave": true,
    "gitIntegration": true,
    "testIntegration": true
  }
}
```
